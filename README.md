# logseq-plugin-luckysheet

在 Logseq 中嵌入电子表格 Luckysheet。你也可以用它来维护一张 markdown 表格。

表格数据存储在 `assets` 文件夹下。

## 使用展示

![demo](demo.gif)

## 使用示例

```
{{renderer :luckysheet, workbook name}}
```
